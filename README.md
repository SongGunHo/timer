
# Timer (Vanilla JavaScript)

간단한 웹 타이머 프로젝트입니다.  
`setInterval`로 시간을 제어하고, DOM 업데이트로 화면에 `HH:MM:SS` 형식으로 표시합니다.

## Demo
- 로컬에서 `index.html` 실행

## Features
- Start / Stop / Reset
- 실시간 시간 표시 (HH:MM:SS)
- 중복 실행 방지(타이머 상태 관리)

## Tech Stack
- HTML5
- JavaScript (Vanilla JS)

├─ index.html
└─ timer.js
